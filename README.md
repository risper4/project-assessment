# Address book website
A modern and technical address book of putting down places visited as well as brief details about the sites visited.

### Author: Risper Gichia

## Project description
Address book website is a modern way of taking note of different places visited along side a brief description of the location, date, landmarks as well as terse notes on your whole experince.This is an efficient and more simpler method of keeping memories for as long as possible.

## Set up instructions
* Head and access the website using the url
* Download site or access the site without logging in as long as you are online
* Key in site visited on the form
* Access the site details once you click on the site mentioned.

## Technologies used
The following are the technologies used:
* HTML
* CSS
* Javascript
